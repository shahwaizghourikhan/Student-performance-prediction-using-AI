

 🎓 AI-Based Student Performance Prediction System

Overview

The AI-Based Student Performance Prediction System is a Machine Learning-based web application that predicts student academic performance using historical educational data. The system analyzes key academic factors such as study hours, attendance percentage, assignment marks, and quiz marks to estimate a student's final grade.

The project is developed using **Python**, **Streamlit**, **Pandas**, **Scikit-learn**, and **Matplotlib**. A **Random Forest Regression** model is trained on student records stored in a CSV dataset to generate accurate grade predictions and identify students who may require additional academic support.

Features

* Predict student final grades using Machine Learning
* View complete student academic records
* Interactive Streamlit dashboard
* Dataset visualization and analysis
* Student performance categorization
* Risk detection for low-performing students
* Model accuracy evaluation using R² Score

Technologies Used

* Python
* Streamlit
* Pandas
* Scikit-learn
* Matplotlib
* CSV Dataset

Machine Learning Model

The project uses Random Forest Regression to predict student final grades based on:

* Study Hours
* Attendance Percentage
* Assignment Marks
* Quiz Marks

Performance Categories
